# 🏠 HomeLab

## 🎯 Objectif
Créer un DNS sécurisé accessible localement et à distance.

## 🧱 Infrastructure

| Composant | Rôle |
|----------|------|
| Raspberry Pi Zero 2 W | Serveur principal |
| Pi-hole | DNS / Adblock |
| Tailscale | VPN sécurisé |
| Freebox | Routeur |

## 🔐 Avantages
- Pas d'ouverture de ports
- Accès sécurisé via WireGuard
- Filtrage DNS global
